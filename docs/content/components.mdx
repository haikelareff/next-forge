# Premium UI Components

Welcome to Hrev Dev's comprehensive collection of premium UI components. Each component is carefully crafted with accessibility, performance, and developer experience in mind.

## Getting Started

Our components are built on top of the popular shadcn/ui library, extending it with advanced functionality and beautiful designs.

### Installation

```bash
npm install @hrevdev/ui
# or
pnpm add @hrevdev/ui
```

### Usage

```tsx
import { AdvancedDataTable } from '@hrevdev/ui/data-table'
import { MultiStepWizard } from '@hrevdev/ui/forms'

export default function MyApp() {
  return (
    <div>
      <AdvancedDataTable data={data} columns={columns} />
      <MultiStepWizard steps={steps} />
    </div>
  )
}
```
